## Fibonacci Sequence

<TulparPlayground
	title="Fibonacci"
	code={String.raw`func fibonacci(int n) {
    if (n <= 1) {
        return n;
    }
    return fibonacci(n - 1) + fibonacci(n - 2);
}

print("Fibonacci sequence:");
for (int i = 0; i < 10; i++) {
    print("F(" + toString(i) + ") =", fibonacci(i));
}`}
/>

## JSON Data Processing

<TulparPlayground
	title="JSON Data Processing"
	code={String.raw`arrayJson users = {
    "data": [
        {"name": "Alice", "age": 25, "role": "Developer"},
        {"name": "Bob", "age": 30, "role": "Designer"},
        {"name": "Charlie", "age": 35, "role": "Manager"}
    ]
};

// Process user data
for (int i = 0; i < length(users["data"]); i++) {
    arrayJson user = users["data"][i];
    str name = user["name"];
    int age = user["age"];
    str role = user["role"];
    
    print(name, "-", age, "years old -", role);
}`}
/>
