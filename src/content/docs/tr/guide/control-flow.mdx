## If-Else İfadeleri

<TulparPlayground
	title="If / Else"
	code={String.raw`int yas = 18;

if (yas >= 18) {
    print("Yetişkin");
} else {
    print("Reşit değil");
}

// Mantıksal operatörler
if (yas >= 18 && yas < 65) {
    print("Çalışma çağında");
}`}
/>

## While Döngüsü

<TulparPlayground
	title="While Döngüsü"

## For Döngüsü

Tulpar hem C tarzı for döngülerini hem de for-each döngülerini destekler:

<TulparPlayground
	title="For Döngüleri"
	code={String.raw`// C tarzı for döngüsü
for (int i = 0; i < 10; i++) {
    print("i =", i);
}

// Range ile for-each
for (i in range(10)) {
    print("i =", i);
}`}
/>
