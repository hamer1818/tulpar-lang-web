## Fibonacci Dizisi

<TulparPlayground
	title="Fibonacci Dizisi"
	code={String.raw`func fibonacci(int n) {
    if (n <= 1) {
        return n;
    }
    return fibonacci(n - 1) + fibonacci(n - 2);
}

print("Fibonacci dizisi:");
for (int i = 0; i < 10; i++) {
    print("F(" + toString(i) + ") =", fibonacci(i));
}`}
/>
