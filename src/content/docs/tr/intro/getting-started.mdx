## Yerel olarak çalıştırma

Programı `merhaba.tpr` adında bir dosyaya kaydedip terminalden
çalıştırın — kurulum sonrası `tulpar` komutu `PATH`'inizdedir:

```bash
tulpar merhaba.tpr
```

Varsayılan `tulpar <dosya>` çağrısı LLVM ile AOT derler ve oluşan native
binary'yi çalıştırır. Anlık başlatma için bytecode VM'i tercih edebilirsiniz
(çok küçük bir runtime maliyeti karşılığında):

```bash
tulpar --vm merhaba.tpr
```

Tulpar toolchain'i olmadan dağıtabileceğiniz tek başına bir native
executable üretmek için:

```bash
tulpar build merhaba.tpr      # → ./merhaba (Windows'ta merhaba.exe)
```

## Etkileşimli REPL

Tulpar'ın bir REPL'i (Oku-Değerlendir-Yazdır Döngüsü) de var.
Argümansız çağırın:

```bash
tulpar --repl
```

Yazdığınız ifadeleri / komutları anında değerlendirir. Çıkmak için
`Ctrl+D` (Linux/macOS) ya da `Ctrl+Z` (Windows) kullanın.
